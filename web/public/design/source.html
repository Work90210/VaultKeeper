CloneHow to get running

One command. Fifteen minutes.

Installs on Debian/Ubuntu LTS, RHEL 9, and OpenSUSE Leap. Requires 4 vCPU, 16 GB RAM, 200 GB storage as an absolute minimum — though most institutions run far more. Runs offline from commit 1.

# Clone and run — single-node, self-contained
$ git clone https://git.vaultkeeper.coop/vaultkeeper/core
$ cd core && ./vk up

→ generating witness keypairs (Ed25519) ✓
→ bootstrapping custody ledger (PostgreSQL 16) ✓
→ provisioning TSA connections (tsa.dfn.de default) ✓
→ loading on-prem AI models (~4.2 GB, one-time) ✓
→ sealing genesis block ✓
